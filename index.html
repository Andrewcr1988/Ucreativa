Que bootstrap?

Bootstrap es una framework de CSS y JavaScript por medio de codigo
predeterminado, que puede ser utilizado en el codigo html para preformatear tu codigo a un estilo predeterminado.

Descagar bootstrap

Facil para el usuario

Un framework muy utilizado por desarrolladores web en cualquier parte y integrante de muchas herramientas en linea

Responsive

Integrado con modelos responsivos para cualquier dispositivo.

Estilos unicos

Con una variedad de estilos puedes crear la base de cualquier pagina web y luego darle tu toque personal.

Estas son las opciones que puedes eligir para trabajar

Tool set 1
Accordion

El acordeón utiliza el colapso internamente para hacerlo plegable. Para renderizar un acordeón expandido, agregue la clase .open en .accordion.

Ejemplo de codigo
Alertas

Las alertas están disponibles para cualquier longitud de texto, así como un botón de cierre opcional. Para un estilo adecuado, use una de las ocho clases contextuales requeridas (por ejemplo, .alert-success). Para el despido en línea, use el complemento de JavaScript de alertas.

Ejemplo de Codigo
Badges

Las insignias se escalan para coincidir con el tamaño del elemento principal inmediato mediante el tamaño de fuente relativo y unidades em. A partir de la v5, las insignias ya no tienen estilos de enfoque o desplazamiento para los enlaces.

Ejemplo de Codigo
Spinners

Los "Spinners" de Bootstrap se pueden utilizar para mostrar el estado de carga en sus proyectos. Están construidos solo con HTML y CSS, lo que significa que no necesita JavaScript para crearlos. Sin embargo, necesitará JavaScript personalizado para cambiar su visibilidad.

Ejemplo de Codigo
Breadcrumbs

Indica la ubicación de la página actual dentro de una jerarquía de navegación que agrega separadores automáticamente a través de CSS.

Ejemplo de Codigo
Pagination

Usamos un gran bloque de enlaces conectados para nuestra paginación, lo que hace que los enlaces sean difíciles de pasar por alto y fácilmente escalables, al mismo tiempo que proporcionamos grandes áreas de impacto.

Ejemplo de Codigo
buttons

Use los estilos de botones personalizados de Bootstrap para acciones en formularios, cuadros de diálogo y más con soporte para múltiples tamaños, estados y más. Incluyendo el boton grupal

Ejemplo de Codigo
Card

Una tarjeta es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y potentes opciones de visualización.

Ejemplo de Codigo
Carrousel

El carrusel es una presentación de diapositivas para recorrer una serie de contenido, construido con transformaciones CSS 3D y un poco de JavaScript. Funciona con una serie de imágenes, texto o marcado personalizado.

Ejemplo de Codigo
Colapse

The collapse JavaScript plugin is used to show and hide content. Buttons or anchors are used as triggers that are mapped to specific elements you toggle.

Ejemplo de Codigo
Drop downs

Los menús desplegables son superposiciones contextuales que se pueden alternar para mostrar listas de enlaces y más. Se hacen interactivos con el complemento JavaScript desplegable Bootstrap incluido. Se alternan al hacer clic, no al pasar el mouse; esta es una decisión de diseño intencional.
